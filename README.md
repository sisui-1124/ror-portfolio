## 概要
Ruby on Railsの勉強として作成した、メディアのポートフォリオです。
一般的なメディアに実装される、基本的な機能を実装しました。

## バージョン
Ruby 2.4.5

Rails 5.2.2

## 実装機能

### 記事関連
- 一覧表示機能
- 詳細表示機能
- 削除機能
- 編集機能
- プレビュー機能
- 人気記事一覧表示機能
- キーワード検索機能
- タグ別記事一覧表示機能


### ユーザー関連
- 登録機能
- 編集機能(アドレス・パスワード)
- 管理ユーザー追加機能
- ログイン・アウト機能

### コメント関連
- 投稿機能
- 削除機能

## その他
- dockerによる環境構築
- rspec、capybaraによる簡易的なテストの導入
- ciツールによるテスト、デプロイ自動化
